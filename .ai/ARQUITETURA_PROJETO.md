# PROJETO: Sistema de Certificação Escolar

## ARQUITETURA
Há dois modos principais em desenvolvimento no sistema:
- O modo de Página Única (Single Page), onde os módulos são apresentados como 'PAINEIS' são colpasáveis/expansíveis dentro de uma mesma página.
- O modo de Múltiplas Páginas, onde os módulos são apresentados em páginas separadas, navegando entre elas.

Para qualquer termo de glossário que envolva banco de dados, considerar a interpretação dos modelos de dados conforme definidos em `prisma/schema.prisma`.
- Se ainda assim permanecer dúvida, pode perguntar fornecendo opções baseadas no que foi observado no arquivo de esquema Prisma e com base em boas práticas de modelagem de dados.
