# PROJETO: Sistema de Certificação Escolar

## ARQUITETURA
Há dois modos principais em desenvolvimento no sistema:
- O modo de Página Única (Single Page), onde os módulos são apresentados como 'PAINEIS' são colpasáveis/expansíveis dentro de uma mesma página.
- O modo de Múltiplas Páginas, onde os módulos são apresentados em páginas separadas, navegando entre elas.

### NOMENCLATURAS

- 'PAINEL' = Modo como os módulos são apresentados em Página Única (Single Page).
- 'PÁGINA' = Modo como os módulos são apresentados em Múltiplas Páginas.
