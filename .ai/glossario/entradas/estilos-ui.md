## GLOSSÁRIO DE ESTILOS DE UI
 
 ### Estilo de UI Ultra-Compacto
 **Categoria**: [Estilo de Interface]
 **Modo de chamada**: `Estilo de UI Ultra-Compacto`
 **Descrição**: Estilo de interface de usuário que maximiza a quantidade de informações exibidas na tela, minimizando espaços em branco e utilizando fontes e elementos gráficos menores.
   - Itens de lista com linha única, espaçamentos mínimos entre elementos, fontes pequenas.
**Definições técnicas**:
 *Tamanho de fonte padrão*: 12px;
 *Espaçamento entre linhas*: 1.2;
 *Margens e paddings*: 4px;

 ### Estilo de UI Compacto
 **Categoria**: [Estilo de Interface]
 **Modo de chamada**: `Estilo de UI Compacto`
 **Descrição**: Estilo de interface de usuário que equilibra a densidade de informações com a legibilidade, utilizando tamanhos de fonte e espaçamentos moderados.
**Definições técnicas**:
 *Tamanho de fonte padrão*: 14px;
 *Espaçamento entre linhas*: 1.4;
 *Margens e paddings*: 8px;
