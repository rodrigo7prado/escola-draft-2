## GLOSSÁRIO DE ESTILOS DE UI
 
 **Estilo de UI Ultra-Compacto**: Estilo de interface de usuário que maximiza a quantidade de informações exibidas na tela, minimizando espaços em branco e utilizando fontes e elementos gráficos menores.
    - Itens de lista com linha única, espaçamentos mínimos entre elementos, fontes pequenas.

 **Estilo de UI Compacto**: Estilo de interface de usuário que equilibra a densidade de informações com a legibilidade, utilizando tamanhos de fonte e espaçamentos moderados.
