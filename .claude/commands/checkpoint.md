# Comando: Atualizar Checkpoint IDD

Você está atualizando os checkpoints da feature atual seguindo a metodologia IDD.

## Passos a executar:

1. **Identificar a feature atual**
   - Ler docs/IDD.md para identificar a FEATURE ATUAL

2. **Localizar arquivo de CHECKPOINT**
   - Localização: docs/[NOME_DA_FEATURE]/CHECKPOINT.md
   - Ler o arquivo completo

3. **Conversar com o usuário:**
   - Perguntar quais checkpoints foram concluídos nesta sessão
   - Confirmar o que deve ser marcado como [x]
   - Perguntar se há novos checkpoints a adicionar

4. **Atualizar CHECKPOINT.md**
   - Marcar checkpoints concluídos com [x]
   - Adicionar novos checkpoints se necessário
   - Manter hierarquia e formato

5. **Atualizar TECNICO.md**
   - Perguntar ao usuário sobre decisões técnicas importantes
   - Adicionar resumo enxuto das motivações técnicas
   - Formato: "T1. Resumo da motivação técnica..."

6. **Resumo final**
   - Listar o que foi marcado como concluído
   - Listar próximos checkpoints pendentes
   - Confirmar se está tudo correto

## Importante:
- Sempre usar português
- Tom acolhedor e colaborativo
- Confirmar cada etapa com o usuário
- Manter formato exato dos arquivos conforme IDD.md
