# Workflow para uso do Claude AI / ChatGPT Codex
- Identificar a feature a ser implementada
- Verificar arquivo de Fluxo
  - a regra para o nome é: [NOME_DA_FEATURE]_FLUXO.md
- Verificar arquivo de CHECKPOINTS no arquivo TECNICO
  - a regra para o nome é: [NOME_DA_FEATURE]_TECNICO.md (haverá a sessão de checkpoints dentro deste arquivo)
- Ao final de cada sessão de trabalho da IA, atualizar o arquivo TECNICO com os checkpoints atingidos;
- Preparar a próxima sessão de trabalho com base nos checkpoints atingidos e os próximos a serem atingidos;
- Repetir até a feature estar completa;
