## DRY.UI
Este diretório documenta os componentes de UI (Interface do Usuário) desenvolvidos para o projeto. A organização dos arquivos segue uma lógica de camadas.

- `ui-base.dry.md`: Documenta os componentes base de UI que são reutilizáveis em toda a aplicação, como botões, modais, ícones, etc.
- `ui-components.dry.md`: Documenta componentes de UI mais complexos que combinam múltiplos componentes base para formar funcionalidades específicas.
- `ui-features.dry.md`: Documenta as montagens de componentes de UI para formar telas ou funcionalidades completas da aplicação.
