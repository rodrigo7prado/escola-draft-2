# FLUXO DE USO
F1: O usuário acessa o Painel de Migração, podendo expandi-lo para visualizar as opções disponíveis.
F2: O usuário clica em "Selecionar arquivos" na seção de Ficha Individual - Histórico.
F2: Abre a janela de seleção de arquivos do sistema operacional.
F3: O usuário confirma a seleção dos arquivos desejados.
F4: O sistema valida os arquivos selecionados.
F5: O sistema exibe o progresso da importação dos arquivos, emitindo as mensagens de status apropriadas.
F6: O sistema conclui a importação e exibe uma mensagem de sucesso ou erro.
F7: O painel de Gestão do Aluno é atualizado com os status da importação para cada aluno (detalhes de implementação futuramente).
F8: O usuário terá condições de visualizar integralmente o status de todas as importações realizadas, com os dados resumidos e bem organizados, tudo em apenas uma tela;
